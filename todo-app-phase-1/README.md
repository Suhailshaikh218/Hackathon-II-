# Todo App

A simple command-line todo application with in-memory storage.

## Features
- Add new tasks
- List all tasks
- Mark tasks as completed
- Delete tasks
- Clear all tasks

## How to Use

1. Run the application:
```bash
python main.py
```

2. Use the menu options:
   - Option 1: Add a new task
   - Option 2: List all tasks
   - Option 3: Mark a task as completed
   - Option 4: Delete a specific task
   - Option 5: Clear all tasks
   - Option 6: Exit the application

## Technical Details

- Built with Python 3.13
- Uses in-memory storage (no database)
- Includes type hints for all functions
- Follows clean code principles
